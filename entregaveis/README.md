# Entregáveis

Esta pasta reúne materiais produzidos pela equipe a partir do contexto e das fontes. Arquivos aqui são derivados e não substituem evidências catalogadas.

## Regras

- Crie uma subpasta por entregável ou finalidade.
- Mantenha o conteúdo-base em Markdown quando aplicável.
- Use número de versão e data no nome do arquivo.
- Preserve a versão efetivamente apresentada ou enviada.
- Identifique claramente `rascunho interno`, `em revisão` ou `aprovado`.
- Nada destinado a terceiros sai sem a revisão prevista no `AGENTS.md`.
